STUDENT MANAGEMENT SYSTEM
a program created to manage, add, search, delete and update students from a local school.
this program is a terminal based program created in Python language.

It contains:
- Dictionary
- Menu
- Add student function
- Show student function
- delete student function
- search student function
- update studion function

Now a days, inventory processes have been replaced by digital data storage programs. This S.M.S will help teachers from a local school to storage student's information instead of writing them down o a piece of paper. In addition to that, student's information can be managed as teachers will. And they do not have to struggle again handling big books to make sure students are registered.
